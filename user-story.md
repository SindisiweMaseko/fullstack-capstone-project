# User Stories for GiftLink

## Epic: Item Management
1. **As a** user looking to declutter, **I want to** list an unwanted household item, **so that** someone else can reuse it instead of throwing it away.
2. **As a** user searching for free items, **I want to** browse all available listings, **so that** I can find things I need.
3. **As a** user, **I want to** view details of a specific item, **so that** I can see its condition and location before requesting it.

## Epic: User Authentication
4. **As a** new user, **I want to** register for an account, **so that** I can securely list items and leave reviews.
5. **As a** returning user, **I want to** log in securely, **so that** I can access my profile and manage my listings.
